## Getting Started
1. Clone this repository
```bash
git clone https://github.com/AryaGoberto/shopora.git
```
2. Navigate to the project directory
```bash
cd shopora
```
4. Install dependencies
```bash
npm install
```
5. Run the application
```bash
npm run dev
```
